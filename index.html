FigDU - Figma Developer Utility

Build Figma plugins effortlessly with FigDU.

Installation

Run the following command to create a new FigDU project

npx create-figdu

or

npm create figdu

Getting Started

Navigate to your project directory:

cd your-figdu-project

Build the project:

npm run build

(In 4 easy Steps)

  • Go to the Plugins menu in Figma.
  • Select Development > Import plugin from manifest.
  • Choose the manifest.json file from your project directory.
  • Start Development.
